About OSCAR Lab

Empowering the Next Generation of Computing Systems

Operating System and Computer ARchitecture Laboratory, NYCU

Welcome to the Operating System and Computer ARchitecture (OSCAR) Lab at NYCU!
We are dedicated to pioneering research in system software, computer architecture, and emerging memory and storage technologies. Our team is passionate about software/hardware co-design with memory and storage systems, memory-centric system design (e.g., near-data processing), and operating system redesign. We actively seek creative minds to join us in advancing high-performance, efficient, and robust next-generation computing solutions.

If you are curious about reimagining the fundamentals of computers, enjoy building real systems, or want to make an impact in AI, storage and memory system. Let's start a conversation!
